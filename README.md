This repository hosts a set of visual educational tools designed to bridge the gap between abstract thermodynamic theories and real-world engineering applications.

## 🔗 Live Gallery
**Explore the interactive cards here:** [https://hzgezi-lynette.github.io/Thermodynamics-teaching-materials/](https://hzgezi-lynette.github.io/Thermodynamics-teaching-materials/)

## 📖 Design Philosophy
Thermodynamics is often perceived as a collection of daunting equations. This project aims to simplify these concepts for undergraduate students by:
- **Visualizing Abstractions:** Turning formulas into intuitive diagrams.
- **Industry-Ready Mindset:** Explicitly linking classroom theory to industrial scenarios (e.g., fuel cells, nuclear power, and heat engines).
- **Categorical Learning:** Grouping knowledge into five distinct themes: Basic Concepts, Energy Flow, Electrochemistry, Functional Analysis, and Nuclear/Radiative Quantities.

## 🗂️ Project Structure
- `index.html`: The interactive web gallery.
- `images/`: Optimized web-version learning cards.
- `compress.py`: Python script used for image optimization and batch processing.
